# Hardware

All physical hardware documentation, materials, and references.

## Subfolders

### `bom/`
Bill of Materials - component lists, supplier links, pricing.
- Place spreadsheets (.xlsx, .csv) or markdown tables here.

### `photos/`
Photos of the physical setup, assembly progress, and reference images.
- Name convention: `YYYY-MM-DD_description.jpg`

### `table_design/`
Table structure engineering drawings and dimensions.
- Acrylic cutting templates
- Aluminum frame specs
- Grid constraint layer drawings
- Assembly instructions

## Key Specs (from docs)

- Table unit: ~60x60 cm (8x8 grid with 6x6 plates)
- Acrylic: 5-8mm transparent
- Camera cavity: 25-40cm under table
- Lighting: 5000K neutral LED
